机器学习——多项式回归

发表于 2020-01-14 | 分类于 机器学习
字数统计: 1,082 字 | 阅读时长 ≈ 4 分钟

【概述】

无论是一元线性回归还是多元线性回归，其局限性在于只能应用于存在线性关系的数据中，但在实际应用里，很多数据间是非线性关系。

虽然可以用线性回归来拟合非线性回归，但效果会十分差，这时就要对线性回归模型进行改进，使其能拟合非线性数据，即多项式回归。

如下图，左边是利用线性回归进行拟合的，右边是利用多项式回归进行拟合的，可以看出，采用多项式回归拟合的效果较好。

【基本思想】

多项式回归是在线性回归基础上进行的改进，相当于为样本添加特征项，来拟合非线性的数据。

举例来说，当我们要预测房屋价格 $y$ 时，给出房屋占地长度 $l$ 和宽度 $w$

当我们利用线性回归的方法，假设函数为 $h_{\theta}(l,w) = \theta_0 + \theta_1 l + \theta_2 w$

利用线性回归的方法来拟合的数据并不一定合适，我们可以利用多项式回归的方法来拟合。

具体来说，我们知道房屋的面积 $S=l*w$，那么我们可以通过给出的房屋占地长度与宽度，将添加一个新的特征值——房屋面积 $S$

考虑到价格可能是一个二次函数，那么假设函数可以为：$h_{\theta}(S) = \theta_0 + \theta_1 S + \theta_2 S^2$

如上图，二次函数拟合的图像先增后降，很明显不合理，随着房屋面积的增长房价不会下跌，反而可能会一直上涨，那么考虑用一个三次函数来进行拟合，那么假设函数就为：$h_{\theta}(S) = \theta_0 + \theta_1 S + \theta_2 S^2 + \theta_3 S^3$

或者，随着房屋面积的增长，房价可能会先进行上涨，然后趋于平稳，那么我们就可以借助平方根函数来进行拟合，假设函数为：$h_{\theta}(S) = \theta_0 + \theta_1 S + \theta_2 \sqrt{S}$

有了多项式方程后，要考虑如何将模型与数据进行拟合，此时只需要将多元线性回归做一个简单的修改。

我们以三次函数举例说明，在多元线性回归时，假设函数为 $h_{\theta}(x) = \theta_0 + \theta_1 x_1 + \theta_2 x_2 + \theta_3 x_3$，我们只需要令 $x_1=S,x_2=S^2,x_3=S^3$，就可以得到多项式回归的形式： $h_{\theta}(S) = \theta_0 + \theta_1 S + \theta_2 S^2 + \theta_3 S^3$

然后，我们再利用梯度下降法来寻找合适的 $\theta$ 参数，需要注意的是，我们构建出来的特征值的范围不一定合适，为了让梯度下降法尽可能的减少迭代次数，需要合理的使用特征缩放来进行处理。

综上所述，多项式回归本质上是对线性回归的一种改进，通过对已有特征值的操作，来组合新的特征值，然后通过对函数图像的理解与数据的实际情况，选择不同的特征，来得到更好的模型。
