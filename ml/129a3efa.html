机器学习——Logistic回归

发表于 2020-01-16 | 分类于 机器学习
字数统计: 2,913 字 | 阅读时长 ≈ 11 分钟

【概述】

逻辑回归（Logistic Regression）是用于解决分类问题的一种分类算法，其在现实中应用十分广泛，比如垃圾邮件识别，手写数字识别，人脸识别，语音识别等。

关于分类问题：点击这里

对于分类问题来说，如果我们利用线性回归来拟合，假设函数 $h_{\theta}(x)$ 的输出值会出现 $h_{\theta}(x)>1$ 或 $h_{\theta}(x)<0$ 的情况，无法进行结果归纳，因此，为了处理分类问题，有了 Logistic 回归，其得到的结果可以满足 $0 \leq h_{\theta}(x) \leq 1$。

Logistic 回归根据分类数据的类别，分为以下三种情况：

  • 二元 Logistic 回归：分类数据为两类（例如：有/没有）
  • 多元无序 Logistic 回归：分类数据超过两类，且类别间没有对比意义（例如：一线城市、二线城市、三线城市）
  • 多元有序 Logistic 回归：分类数据超过两类，且类别间有对比意义（例如：喜欢、不喜欢、无所谓）

其中，二元 Logistic 回归是最基础的 Logistic 回归，对于输出 y，其用 1 表示正向类，用 0 表示负向类，即有：$y \in \{ 0,1 \}$，近似地，多元 Logistic 回归会有更多的类别，即：$y \in \{ 0,1,2,3,… \}$

【假说表示】

简单来说，假说表示就是当有一个分类问题的时候，我们要使用什么样的方程来表示我们的假设。

分类器

在 Logistic 回归中，分类器即假设函数 $h_{\theta}(x)$

我们首先对 Logistic 回归的分类器进行解释：$h_{\theta}(x)$ 表示当输入为 x 时，$y=1$ 的概率估计。

假设分类器只有一个特征值 $x$，其代表的是肿瘤的大小，这时有一个病人来了，他的肿瘤是某个大小 $x=sumorSize$，将这个特征值带入分类器中，并假设输出为 $h_{\theta}(x)=0.7$

那么这个假设我们可以这样解释：对于一个特征为 $x$ 的患者，$y=1$ 的概率是 0.7，换句话说，这个肿瘤有 70% 的可能是一个恶性肿瘤。

假说表示

由于我们希望我们的分类器的输出值在 0 和 1 之间，因此我们需要一个满足 $0 \leq h_{\theta}(x) \leq 1$ 的假设函数。

在线性回归中，我们有假设函数：$h_{\theta}(x)=\theta^Tx$

而对于 Logistic 回归，我们对线性回归进行修改，为其加上一个 激活函数 $g(z)$ ，使得能够将权值结果转换为分类结果，即：$h_{\theta}(x)=g(\theta^Tx)$

这个激活函数 $g(z)$，就是我们的假说表示，对于不同的问题，我们根据实际情况来选择不同的激活函数。

【决策边界】

决策边界（Decision Boundary）是假设函数的一个属性，其决定于 $\theta$ 参数，一旦通过训练集拟合出了 $\theta$ 参数，就确定了决策边界。

我们首先来看假设函数：$h_{\theta}(x)=g(\theta^TX),g(z)=\frac{1}{1+e^{-z}}$

我们给出 sigmoid 函数的图像，来看一下假设函数何时会预测 $y=0$，何时又会预测 $y=1$

根据 sigmoid 函数图像的对称性，我们可以认为，只要假设函数输出 $y=1$ 的概率大于或等于 0.5，那么意味着 $y=1$，相反地，只要假设函数输出 $y=0$ 的概率小于 0.5，那么意味着 $y=0$，即：

根据 sigmoid 函数的图像，可以看出，只要 $z \geq 0$，就有 $g(z) \geq 0.5$，也就是说，只要 $\theta^TX \geq 0$，那么 $h_{\theta}(x) \geq 0.5$，我们就预测 $y=1$，同理，只要 $\theta^TX < 0$，那么 $h_{\theta}(x) < 0.5$，我们就预测 $y=0$

简单来说，如果我们要预测 $y=1$，只需要 $\theta^TX \geq 0$，如果我们要预测 $y=0$，只需要 $\theta^TX < 0$

决策边界，是一条线，其将预测值为 1 和 0 的两个区域分隔开来，这一条线对应的就是 $h_{\theta}(x)=0.5$ 即 $\theta^TX=0$ 时对应的线。

如下图，假设我们有一个模型：$h_{\theta}(x)=g(\theta_0+\theta_1x_1+\theta_2x_2)$，通过训练集拟合出的 $\theta$ 参数的转置 $\theta^T=[-3,1,1]$

当 $\theta^TX \geq 0$，即：$-3+x_1+x_2 \geq 0$ 时，模型预测 $y=1$

当 $\theta^TX < 0$，即：$-3+x_1+x_2 < 0$ 时，模型预测 $y=1$

此时，$x_1+x_2=3$ 就是该模型的决策边界。

【代价函数】

引入

在线性回归模型中，我们的代价函数是 $J(\theta) = \frac{1}{2n}\sum_{i=1}^n(h_{\theta}(x^{(i)}) - y^{(i)})^2$

在 Logistic 回归中，我们将假设函数 $h_{\theta}(x)=\frac{1}{1+e^{-\theta^TX}}$ 带入上述的代价函数里，并画出代价函数的图像，会发现可能是一个类似下图的非凸函数，即有许多局部最优值，如果将梯度下降法用在这样一个函数上，无法保证其能收敛到全局最小值。

因此，为了保证使用梯度下降法使得代价函数收敛到全局最小值，代价函数需要是一个凸函数，那么我们就要另找一个不同的代价函数。

单项样本的代价函数

我们首先考虑单样本时的代价函数 $Cost(h_{\theta}(x),y)$

【多类别分类】

多类别分类问题是指分类数据超过两类时的分类问题，举例来说，如果我们需要将邮件自动归类到四个不同的文件夹中，这时我们就有了一个包含四个分类的问题，此时，输出 $y$ 可表示为：$y \in \{ 1,2,3,4 \}$

下面，我们以一个三类别分类问题为例，来说明多类别分类问题的模型拟合方法。

原理

假设训练集数据如下图，其包含三个类别

我们要做的，就是将这个三元分类问题转换成三个独立的二元分类问题。

我们从类别 1 开始依次创建三个伪训练集：

  • 类别 1 为正类，类别 2 与 类别 3 为负类
  • 类别 2 为正类，类别 1 与 类别 3 为负类
  • 类别 3 为正类，类别 1 与 类别 2 为负类

然后对这三个伪训练集分别创建分类器：$h_{\theta}^{(1)}(x),h_{\theta}^{(2)}(x),h_{\theta}^{(3)}(x)$，即：

之后，我们利用 Logistic 回归来拟合参数 $\theta^{(1)},\theta^{(2)},\theta^{(3)}$，从而得到三个决策边界。

也就是说，每个分类器都针对其中一种情况进行训练：

  • 当 $i=1$ 时，$h_{\theta}^{(i)}(x)$ 计算的是 $y=1$ 的概率是多少
  • 当 $i=2$ 时，$h_{\theta}^{(i)}(x)$ 计算的是 $y=2$ 的概率是多少
  • 当 $i=3$ 时，$h_{\theta}^{(i)}(x)$ 计算的是 $y=3$ 的概率是多少。

最后，为了做出预测，当我们输入新的特征值 $x$ 后，在这三个分类器中分别运行输入 x，然后选择最大的 $h_{\theta}^{(i)}(x)$ 的值，即可信度最高、效果最好的分类器，此时得到的概率值，就是要预测的 $y$ 值。
