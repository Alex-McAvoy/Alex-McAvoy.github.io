机器学习——多元线性回归

发表于 2020-01-13 | 分类于 机器学习
字数统计: 1,444 字 | 阅读时长 ≈ 5 分钟

【概述】

在回归分析中，如果有两个及以上的自变量，且因变量和自变量之间是线性关系，则称为多元线性回归。在实际应用中，一种现象常常是与多个因素相联系的，由多个自变量的最优组合共同来预测或估计因变量，比只用一个自变量的一元线性回归进行预测或估计更有效，更符合实际。

在一元线性回归模型 $h(x)=\theta_0 + \theta_1 x$ 中，我们只有一个特征值 $x$，通过利用批量梯度下降法(BGD) 对代价函数 $J(\theta_0,\theta_1)$ 最小化，来拟合一条直线来描述数据间的关系，这样当出现新的数据时，我们能预测一个 $y$ 值。

而多元线性回归模型，简单来说，就是当有多个特征值 $x_1,x_2,…,x_n$ 时，通过对假设函数来描述数据间的关系，这样当出现新的数据时，就能预测出一个值。

【假设形式】

假设给定 $n$ 个样本，具有 $m$ 个特征值 $x_1,x_2,…x_m$，输出为 $y$，同时，我们将第 $i$ 组样本的所有特征量记为 $x^{(i)}$，输出记为 $y^{(i)}$，将第 $i$ 组样本的第 $j$ 个特征量记为 $x_j^{(i)}$

则假设函数为 $h_{\theta}(x) = \theta_0 + \theta_1 x_1 + \theta_2 x_2 + … + \theta_m x_m$

为了表述方便，我们对假设函数进行简化，假设：$x_0^{(i)}=1$，即定义了一个额外的第 0 个特征量，这个特征量对所有样本的取值全部为 1，这使得特征量从过去的 m 个变为 m+1 个。

那么假设函数就可以写为：$h_{\theta}(x) = \theta_0 x_0+ \theta_1 x_1 + \theta_2 x_2 + … + \theta_m x_m$

【特征缩放】

特征缩放是梯度下降的实用技巧之一，其目的是为了让梯度下降收敛时迭代次数更少。

对于一个机器学习问题，如果该问题有多个特征，且这些特征不同特征的取值都处在一个相近的范围内，这样利用梯度下降就能更快地收敛。

如下图，当特征不在一个相近范围内，使用梯度下降法需要迭代多次次才能找到合适的参数，而利用特征缩放后，只需要迭代很少的次数，即可找到合适的参数。

特征缩放，是将特征的取值约束到接近 $[-1,1]$ 的范围内。

对于 $x_0^{(1)}=1$，其已在这个范围内，我们不作考虑。而对于其他的特征值，我们需要通过除以不同的数，来让它们在同一范围内。类似地，如果特征在一个十分小的范围内，我们也需要除以不同的数，来让它们在同一范围。
