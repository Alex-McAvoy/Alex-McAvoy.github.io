机器学习——一元线性回归

发表于 2020-01-12 | 分类于 机器学习

【概述】

当只有一个自变量和一个因变量，且二者的关系可用一条直线近似表示时，这种回归分析称为一元线性回归分析，这样当出现新的数据时，就能预测出一个值。

如下图，就是一个典型的一元线性回归的例子：给定房屋面积和房价的数据，利用线性回归来构造出一条线来近似地描述房价和面积间的关系，从而根据给出的房屋面积来推测房价。

【假设形式】

一元线性回归模型

通过一元线性回归分析构造出来的只有一个自变量和因变量的模型一般称为一元线性回归模型，其一般形式如下：给定 $n$ 个样本，输入为 $x$，输出为 $y$，同时，将第 $i$ 组样本的所有特征量记为 $x^{(i)}$，输出记为 $y^{(i)}$

则假设函数为 $h_{\theta}(x)=\theta_0 + \theta_1 x$ 。

通过线性回归算法，我们会得到很多的线性回归模型，但是不同的模型对于数据的拟合（描述能力）是不一样的。我们的目的最终是需要找到一个能够最精确地描述数据之间关系的线性回归模型。

因此，我们要引入代价函数的概念。

代价函数

若要使预测更为精确，那么预测就要以现有的数据为根基，尽量的贴合现有数据，使得预测值与真实值的差距尽量小。

这就需要一个代价函数来作为衡量预测结果的指标，在大多数回归问题中一般都选用 均方误差（MSE，Mean Squared Error） 作为代价函数。

我们的目标，就是要令这个代价函数值最小化，而此时的 $\theta_0$ 与 $\theta_1$ 就是预测最精确时假设函数的 $\theta_0$ 与 $\theta_1$。

首先给出平方和误差：$\sum_{i=1}^{n} (h(x^{(i)}) - y^{(i)} )^2$，这里的 $h(x^{(i)}) - y^{(i)}$ 代表这第 $i$ 个样本的预测值 $h(i)$ 与真实值 $y^{(i)}$ 的差值。

这里之所以利用平方和来作为误差函数，而不使用和作为误差函数，是因为 和 会导致正负相消，不能正确的反映出误差。

进一步，我们给出平均平方和误差 即 均方误差 MSE 的概念：$MSE = \frac{1}{n} \sum_{i=1}^{n} (h(x^{(i)}) - y^{(i)} )^2$，其是预测值与真实值的平方和的平均值。

而为了求出代价函数最小化时的 $\theta_0$ 与 $\theta_1$，我们常常利用梯度下降法进行求梯度，由于 MSE 中带了平方，这样求导时会带来 $*2$ 。

为简化计算，一般令 $J(\theta_0,\theta_1) = \frac{1}{2n} \sum_{i=1}^{n} (h(x^{(i)}) - y^{(i)} )^2$ ，这样由求导带来的 $*2$ 就与 $\frac{1}{2n}$ 相抵消，从而简化计算。
