Hexo——LaTeX 渲染

发表于 2020-04-23 | 分类于 Hexo

【插件】

Hexo 的 markdown 渲染引擎是 hexo-renderer-marked，不支持 LaTeX 语法

要想使用 LaTeX，则需要更换渲染引擎，hexo-renderer-kramed 引擎是在默认的渲染引擎 hexo-renderer-marked 的基础上构建的渲染引擎，两者比较接近。
