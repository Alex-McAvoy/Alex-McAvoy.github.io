Matplotlib——概述

发表于 2019-01-02 | 分类于 Python ， Matplotlib
字数统计: 762 字 | 阅读时长 ≈ 3 分钟

【概述】

Matplotlib 是 Python 的绘图库，其可以与 Numpy 一起使用，常用于处理图片、绘制统计类的图表

其通过 pip 进行安装：

安装完成后，通过 pip show matplotlib 可以查看版本号、依赖、安装路径等信息

【基本组成】

Matplotlib 官网上给出下图说明了 Matplotlib 图表的基本组成部分

需要着重区分的如下：

  • figure：整个画布，包含一到多个 axes
  • axes：画布中的某个图表，包含一个 plot
  • artist：figure 上的元素，包含 plot、line、title、label 等

不可交互模式

在非交互模式下，在后台绘制完成的图形是不可交互型，需要使用 show() 函数将图形展现出来

如果想要将图片保存，在 show() 前使用 savefig("路径") 函数即可

例如：

常用的不可交互型有：

  • AGG：渲染为 png 文件
  • PS：渲染为 ps 文件
  • PDF：渲染为 pdf 文件
  • SVG：渲染为 svg 文件
  • Cairo：使用 Cairo 引擎渲染

可交互模式

在可交互模式下，生成的图形是可交互型，即在绘制后可以根据预设或用户操作动态变化，如果希望手动更新图表，需要使用 draw() 函数

通过以下命令可以进行交互模式的设置：

  • matplotlib.is_interactive()：查询当前实发支持交互模式
  • matplotlib.interactive()：设置交互模式的开启与关闭
  • matplotlib.pyplot.ion() 与 matplotlib.pyplot.ioff()：开启/关闭交互模式

常用的可交互型有：

  • Qt5Agg：使用 Qt5 渲染
  • ipympl：使用 ipympl 库渲染
  • nbAgg：Jupyter Notebook 中使用的 backend，在 Jupyter 中使用 %matplotlib notebook 来激活
