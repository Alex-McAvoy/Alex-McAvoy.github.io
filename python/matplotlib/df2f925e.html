Matplotlib——箱线图

发表于 2019-01-25 | 分类于 Python ， Matplotlib
字数统计: 496 字 | 阅读时长 ≈ 2 分钟

【箱线图介绍】

箱线图（Box Plot），又称为盒式图、箱形图，是一种显示一组数据分散情况资料的统计图，常用于品质管理，快速识别异常值等

箱形图最大的优点就是不受异常值的影响，能够准确稳定地描绘出数据的离散分布情况，同时也利于数据的清洗

如上图，箱线图有五大因数：下分位数 Q1、中位数 Q2、上分位数 Q3、上限、下限

其中，下分位数、中位数、上分位数统称为四分位数（$Q_i$），其是将所有数值由小到大排列并分成四等份，处于三个分割点位置的数值，每部分包含 25% 的数据，$Q_i$ 所在的位置为 $\frac{i*(n+1)}{4}$

上限是非异常范围内的最大值，下限是非异常范围内的最小值，其值的计算是通过四分位距来计算的

四分位距（IQR）用于确定第三四分位数（Q3）与第一四分位数（Q1）的区别，以表示统计资料中各变量分散情形，其计算公式为：$IQR=Q_3-Q_1$

在有了四分位距（IQR）后，可以根据其值来计算上限与下限：

  • $上限=Q_3+1.5*IQR$
  • $下限=Q_1-1.5*IQR$

在计算出五大因数后，即可很清晰的找出异常值

【boxplot()】

在 Matplotlib 中，使用 boxplot() 函数来绘制箱线图，其基本语法如下：

plt.boxplot(x, positions=None, vert=True, whis=1.5, usermedians=None, label=None)

其中，各参数如下：

  • x：参数列表
  • positions：每个箱的位置
  • vert：图表方向，默认 True 为垂直，False 为水平
  • whis：上下限与四分位距的关系，默认为计算公式的 1.5
  • usermedians： 使用该参数中的数据代替 x 中计算得出的均值
  • label： 图例显示内容
