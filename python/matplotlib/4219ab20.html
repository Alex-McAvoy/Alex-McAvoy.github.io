Matplotlib——点线图

发表于 2019-01-03 | 分类于 Python ， Matplotlib
字数统计: 825 字 | 阅读时长 ≈ 4 分钟

【plot()】

plot() 函数用于绘制折线图，其基本语法为：

lines=plt.plot(x1,y1,'样式1',x2,y2,'样式2',...)

返回值 lines 是一个列表，其中每个元素是绘制的曲线，样式是可选的，若未设置则采用默认颜色

【loglog()】

loglog() 函数同样用于绘制折线图，其与 plot() 函数的区别是其 x 轴、y 轴是指数型的

其有两个重要的参数 basex、basey，分别用于设置 x 轴、y 轴的底

【semilogx()与 semilogy()】

semilogx() 函数用于绘制指数函数折线图，其 x 轴是指数型，y 轴是线型，semilogy() 函数用于绘制对数函数折线图，其 x 轴是线型，y 轴是指数型

分别有一个重要参数：

  • basex：x 轴的底
  • basey：y 轴的底

【step()】

semilogx() 函数用于绘制阶梯型折线图，其有一个重要参数 where，用于指出数据点位于阶梯的位置：

  • pre：数据点位于阶梯的左侧
  • mid：数据点位于阶梯的正中
  • post：数据点位于阶梯的右侧

【plot_date()】

plot_date() 函数用于绘制时序图，可以将横/纵坐标当做 dates 类型进行处理

其有三个重要参数：

  • xdate：是否将 x 轴作为 dates 类型处理，默认为 True
  • ydate：是否将 y 轴作为 dates 类型处理，默认为 False
  • linestyle：样式，默认为 None，因此需要进行样式设置，否则绘制出的图形像是散点图

【关于样式】

标记

常用的标记格式化字符如下：

  • ‘-‘：实线样式
  • ‘—‘：短横线样式
  • ‘-.’：点划线样式
  • ‘:’：虚线样式
  • ‘.’：点标记
  • ‘,’：像素标记
  • ‘o’：圆标记
  • ‘v’：倒三角标记
  • ‘^’：正三角标记
  • ‘<’：左三角标记
  • ‘>’：右三角标记
  • ‘1’：下箭头标记
  • ‘2’：上箭头标记
  • ‘3’：左箭头标记
  • ‘4’：右箭头标记
  • ‘s’：正方形标记
  • ‘p’：五边形标记
  • ‘*’：星形标记
  • ‘h’：六边形标记 1
  • ‘H’：六边形标记 2
  • ‘+’：加号标记
  • ‘x’：X 标记
  • ‘D’：菱形标记
  • ‘d’：窄菱形标记
  • ‘|’：竖直线标记
  • ‘_’：水平线标记

颜色

常用的颜色缩写如下：

  • ‘b’：蓝色
  • ‘g’：绿色
  • ‘r’：红色
  • ‘c’：青色
  • ‘m’：品红色
  • ‘y’：黄色
  • ‘k’：黑色
  • ‘w’：白色
