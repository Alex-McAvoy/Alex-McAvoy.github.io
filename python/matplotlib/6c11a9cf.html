Matplotlib——坐标轴

发表于 2019-01-11 | 分类于 Python ， Matplotlib
字数统计: 1,406 字 | 阅读时长 ≈ 7 分钟

【axis()】

plt.axis() 用于设置坐标轴的属性，其用法见下：

  • ls = axis()：返回当前坐标轴范围 [xmin, xmax, ymin, ymax]
  • axis([xmin, xmax, ymin, ymax])：设置当前坐标轴范围
  • axis('off')：不显示坐标轴与坐标轴名称
  • axis('equal')：使 x 轴、y 轴对应长度所表示的数值相同，提高某一个轴的范围以保持图表大小及比例不变
  • axis('scaled')：使 x 轴、y 轴对应长度所表达的数值相同，保持两个轴的数值范围不变，并对图表比例进行调整
  • axis('tight')：在显示所有数据的前提下，尽量减少两个轴的数值范围，并尽量让数据居中
  • axis('image')：将图表比例按照图片的比例进行调整
  • axis('square')：将图表变成正方形，并确保 x 轴与 y 轴范围相同
  • axis('auto')：恢复自动范围

【xlabel() 与 ylabel()】

plt.xlabel() 与 plt.ylabel() 分别用于设置 x 轴与 y 轴的标签

以 plt.xlabel() 为例说明该组函数的用法：plt.xlabel(label, labelpad=None, fontsize=12, color=None, sytle='normal')

其中，各参数含义如下：

  • label：标签内容
  • labelpad：标签与 x 轴的距离
  • fontsize：字体大小
  • color：字体颜色
  • style：字体样式
    • 'normal'：正常样式
    • 'italic'：斜体
    • 'oblique'：字体倾斜

【xlim() 与 ylim()】

plt.xlim() 与 plt.ylim() 分别用于设置 x 轴与 y 轴的范围

以 plt.xlim() 为例说明该组函数的用法：

  • maxium = plt.xlim()：返回 (xmin, xmanx)，其中 xmin 为 x 轴的最大值，xmax 为 x 轴的最小值
  • plt.xlim(xmin, xmax)：设置 x 轴的最小值 xmin 与 x 轴的最大值 xmax
  • plt.xlim(xmin=n)：设置 x 轴的最小值 xmin
  • plt.xlim(xmax=n)：设置 x 轴的最大值 xmin

【次坐标刻度】

plt.minorticks_off() 与 plt.minorticks_on() 分别用于关闭、显示当前图表的次坐标刻度，默认处于关闭状态

【xticks() 与 yticks()】

plt.xticks() 与 plt.yticks() 分别用于设置 x 轴与 y 轴的数值范围

以 plt.xticks() 为例说明该组函数的用法：

  • plt.xticks()：获取当前坐标轴的次坐标刻度范围（plt.xlim() 返回的是作图范围）
  • plt.xticks(locs)：设置每个刻度的间距，locs 通常利用 range() 函数来规定
  • plt.xticks(locs, labels)：设置坐标轴的刻标与刻度标签
    • locs：列表类型，刻标的位置
    • labels：列表类型，刻标 loc 对应的标签
  • plt.xticks(locs, labels, rotation=None)：设置坐标轴的刻标与刻度标签，locs 与 labels 同上，rotation 为旋转角度

说明：当不想显示坐标轴刻度时，可以利用 plt.xticks([])

【xscale() 与 yscale()】

plt.xscale() 与 plt.yscale() 分别用于设置 x 轴与 y 轴的缩放方式

以 plt.xscale() 为例说明该组函数的用法：plt.xscale(scale, **kwargs)

其中，scale 可选择如下参数：

  • 'linear'：线性方式，默认
  • 'log'：指数方式，支持的 kwargs 包括：
    • basex 与 basey：x 轴与 y 轴指数的底
    • nonposx 与 nonposy：如何处理负数，mask 直接截掉，clip 化成一个接近 0 的数
    • subx 与 suby：整数列表，确定坐标轴上 subtick 的位置
  • 'symlog'：对称对数类型，支持的 kwargs 包括：
    • basex 与 basey：x 轴与 y 轴对数的底
    • linthreshx 与 linthreshy：x 轴与 y 轴在 $(-n, n)$ 范围内采用线性方式
    • subx 与 suby：整数列表，用于确定坐标轴上 subtick 的位置
  • 'logit'：评定模型，支持的 kwargs 包括：
    • nonpos：如何处理大于 1 或小于 0 的数，mask 直接截掉，clip 化成一个接近 1 或 0 的数
