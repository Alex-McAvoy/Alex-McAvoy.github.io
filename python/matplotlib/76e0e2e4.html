Matplotlib——样式

发表于 2019-02-05 | 分类于 Python ， Matplotlib
字数统计: 460 字 | 阅读时长 ≈ 2 分钟

【默认样式】

Matplotlib 会在以下位置搜索名为 matplotlibrc 的文件：

  1. 当前工作目录
  2. $MATPLOTLIBRC 定义的文件或文件夹内
  3. 用户自定义的配置文件（可利用 matplotlib.get_configdir() 查询），通过 $MPLCONFIGDIR 环境变量设置
  4. 安装目录/matplotlib/mpl-data/matplotlibrc

一旦找到就会停止搜索并使用搜索到的文件内定义的样式作为默认样式

可通过 matplotlib.matplotlib_fname() 查询当前使用的默认样式文件

【样式使用】

查询可用样式

使用 plt.style.available 可以查询当前可用的样式列表，以选择自己所需的样式

样式使用

通过 plt.style.use() 来使用某种样式，参数可以是一个 URL 或路径，也可以是一个列表

自定义样式使用

若不想使用预设的样式，可以将自己的 mplstyle 文件放到 mpl_configdir/stylelib 文件下，这样可以通过文件名称来使用定义的样式，其中 mpl_configdir 可通过 matplotlib.get_configdir() 查询

当使用列表为参数时，可以整合多个 mplstyle 中的样式

【自定义样式参数 rcParams】

matplotlib.rcParams 是一个全局字典，其中存放了大量的样式定义，可以直接进行修改

在 Matplotlib—概述 中，介绍了使用以下方法进行中文显示，其实质上就是对 rcParams 进行设置

若想对其进行永久更改，可通过 matplotlib.matplotlib_fname() 查询当前使用的默认样式文件，对其进行设置
