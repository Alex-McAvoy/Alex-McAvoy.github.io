Matplotlib——热图

发表于 2019-02-07 分类于 Python ， Matplotlib 阅读次数：
本文字数： 1.2k 阅读时长 ≈ 1 分钟

【概述】

热图（Heatmap）是通过密度函数进行可视化来表示点密度的热图，使人们能够独立于缩放因子感知点的密度

在 Matplotlib 中，常利用用于显示图像的 plt.imshow() 函数和显示色阶的 plt.colorbar() 来绘制热图

关于 colorbar()，详见：点击这里

【imshow()】

plt.imshow() 函数用于将数据列表矩阵绘制成图像

plt.imshow(X, **kwargs) 的常见参数如下：

  • X：图像参数，数据为实数或整数
    • [n, m]：二维整型或浮点型矩阵，默认为灰度映射
    • [n, m, 3]：三维浮点型矩阵，第三维范围是 (0,1)，默认为 RGB 映射
    • [n, m, 4]：四维浮点型矩阵，第三维和第四维范围是 (0,1)，默认为 RGBA 映射
  • aplha：透明度
  • cmap：颜色映射方式，详见 colorbar 中的介绍
  • orgin：图片起始位置
    • upper：左上角开始
    • lower：右下角开始
  • aspect：宽高比
    • auto：适应 axes 来调整宽高比
    • equal：axes 适应图像大小来调整宽高比

【热图】

在使用 imshow() 选择好颜色映射方式 cmap 生成图像后，利用 colorbar() 选择同样的颜色映射方式，或将 imshow() 的句柄传入，即可生成热图
