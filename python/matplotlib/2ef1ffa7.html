Matplotlib——常见分布图

发表于 2019-01-21 | 分类于 Python ， Matplotlib
字数统计: 942 字 | 阅读时长 ≈ 4 分钟

【杆图】

plt.stem() 函数用于绘制杆图，其基本语法如下：

ls = plt.stem(x, y, linefmt=None, markerfmt=None, basefmt=None, label=None)

其中，各参数如下：

  • x：横坐标数组，可选
  • y：纵坐标数组
  • linefmt：连线格式
  • markerfmt：标记点格式
  • basefmt： 基线格式
  • label： 图例显示内容

ls 为返回值列表，ls[0]、ls[1]、ls[2] 分别代表杆末端标记点 markerline、杆连线 stemlines、基线的属性 baseline

【电泳图】

plt.eventplot() 用于绘制电泳图，其基本语法如下：

plt.eventplot(positions, orientation='horizontal', lineoffsets=1, linelengths=1, linewidths=None, colors=None, linestyles='solid')

其中，各参数如下：

  • positions：一维或二维列表，每一行都是一组数据，每组数据中各个成员是数据线的位置
  • orientation：分布的方向，可选 horizontal 或 vertical，默认方向为水平
  • lineoffsets：每组数据的偏移位置
  • lineleghths：每条数据线的长度
  • linewidths： 每条数据线的宽度
  • colors： 每条数据线的颜色
  • linestyles：每条数据线的样式，默认为 solid

【间断条形图】

plt.broken_barh() 用于绘制间断条形图，其基本语法如下：

plt.broken_barh(xranges, yrange, facecolors=None)

其中，各参数含义如下：

  • xranges：由 (xmin, xwidth) 组成的列表，xmin 是每个块左端的 x 坐标，xwidth 是每个块的宽度
  • yrange：由 (ymin, ymax) 组成的列表，规定某一组块的 y 坐标范围
  • facecolors：规定每一块的颜色

【errorbar()】

errorbar() 函数用于绘制误差棒图，描述了 x 轴、y 轴方向的误差范围，其基本语法为：

plt.errorbar(x, y, yerr=None, xerr=None, fmt='', ecolor=None, elinewidth=None, ms=None, mfc=None, mec=None, capsize=None, capthick=None)

其中：

  • xerr、yerr：代表误差程度
    • 如果是一个数字，则所有数据点的误差是一个统一的数字
    • 如果是一个一维的数组，则数组表明了每个数据点正向、负向的误差程度
    • 如果是一个二维的数组，则数组表明了每个数据点正向、负向各自的误差程度
  • fmt：数据点的标记样式及之间的连接样式
  • ecolor：误差棒颜色
  • elinewidth：误差棒粗细
  • ms: 数据点的大小
  • mfc: 数据点的颜色
  • mec: 数据点边缘的颜色
  • capsize：误差棒边界横杠大小
  • capthick：误差棒边界横杠厚度
