Matplotlib——六边形分箱图

发表于 2019-01-28 | 分类于 Python ， Matplotlib
字数统计: 456 字 | 阅读时长 ≈ 2 分钟

【概述】

hexbin() 用于绘制六边形分箱图（Hexagonal Binning），是具有六边形单元的二维直方图的伪色彩图（pcolor），根据数据坐标的分布通过颜色的亮暗来显示图像，基本语法如下：

plt.hexbin(x, y, C=None, gridsize=100, bins=None, xscale="linear", yscale="linear", mincnt=None, camp=None)

其中，各参数含义如下：

  • x 与 y：两个一维列表，将根据其组成坐标的分布，绘制分级图
  • C：一维列表，C[i] 会规定 (x[i], y[i]) 坐标的权重，绘制时，某个坐标点的权重会进行平均处理，默认为 None，代表权重为 1 且结果不做平均值处理
  • gridsize：x 轴方向六边形的数量，默认为 100，y 轴方向的六边形数量会自动进行调整
  • bins：分级的方式
    • None：默认方式，六边形的颜色即分级数量
    • "log"：按照 $log_{10}(i+1)$ 决定六边形颜色
    • 整数：按该整数决定等级数量
    • 整数序列：按该整数序列确定每个分级的底线
  • xscale 与 yscale：x 轴、y 轴的坐标轴，默认为 linear 为线性坐标，log 为对数坐标
  • mincnt：不显示数量小于该值的六边形
  • cmap：颜色参数
