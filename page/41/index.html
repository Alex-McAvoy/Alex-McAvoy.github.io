Matplotlib——坐标轴

发表于 2019-01-11 分类于 Python ， Matplotlib
本文字数： 3.8k 阅读时长 ≈ 3 分钟

【axis()】

plt.axis() 用于设置坐标轴的属性，其用法见下：

  • ls = axis()：返回当前坐标轴范围 [xmin, xmax, ymin, ymax]
  • axis([xmin, xmax, ymin, ymax])：设置当前坐标轴范围
  • axis('off')：不显示坐标轴与坐标轴名称
  • axis('equal')：使 x 轴、y 轴对应长度所表示的数值相同，提高某一个轴的范围以保持图表大小及比例不变
  • axis('scaled')：使 x 轴、y 轴对应长度所表达的数值相同，保持两个轴的数值范围不变，并对图表比例进行调整
  • axis('tight')：在显示所有数据的前提下，尽量减少两个轴的数值范围，并尽量让数据居中
  • axis('image')：将图表比例按照图片的比例进行调整
  • axis('square')：将图表变成正方形，并确保 x 轴与 y 轴范围相同
  • axis('auto')：恢复自动范围
